# **App Name**: Monitor do BEEM

## Core Features:

- Class & Student Enrollment: Effortlessly create and manage multiple classes, and add, edit, or remove students for organized class rosters.
- Bloom's Taxonomy Assessment Tool: An intuitive interface to assess student competencies and skills, categorizing progress according to Bloom's Taxonomy levels.
- Student Progress Dashboard: Visualize individual and class-wide learning evolution over time through a clear, graphical dashboard, highlighting trends in competency development.
- Attendance & Observation Logger: Quickly record daily attendance (presence, absence, percentage) and log detailed qualitative observations for each student.
- Dynamic Student Reports: Generate comprehensive student profiles and reports combining assessment results, attendance records, and personalized observations.
- AI Learning Insight Assistant: An AI tool that synthesizes assessment data and observational notes to draft personalized learning suggestions or progress summaries for students.
- Lesson Schedule & Content Planner: Create and manage a class calendar to plan lessons and associate learning content for a clear overview of the curriculum.

## Style Guidelines:

- The color scheme is predominantly light, chosen to promote clarity and focus essential for an educational application. The primary color is a deep, professional blue (#245FB3), reflecting reliability and intelligence.
- The background color is a very light, almost off-white, tone (#ECF1F5). Its subtle blue tint provides continuity with the primary color while maintaining high contrast for text readability.
- An accent color of bright, clean cyan (#33DBDB) is used to draw attention to interactive elements and important notifications, ensuring calls to action stand out clearly.
- All text will use 'Inter', a grotesque-style sans-serif font. Its modern, machined, and objective design ensures high readability and a professional appearance for both headlines and detailed body text, aligning with the app's functional and high-quality aesthetic.
